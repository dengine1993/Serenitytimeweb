# Руководство по эксплуатации

Этот документ описывает три простых шага, которые поддерживают надёжность проекта. Все команды запускаются без глубокой технической подготовки.

## 1. Проверка авторизации в edge‑функциях

Скрипт `scripts/check-auth.ts` проходит по всем функциям Supabase и убеждается, что там используется `getUserFromRequest`.

```bash
npm run lint:auth
```

Если появится сообщение о нарушении, достаточно передать его разработчику (в выводе будет указана конкретная функция).

## 2. Интеграционные тесты edge‑функций

Мы используем Deno‑тесты, которые проверяют три основных сценария: без токена (401), чужой токен (403) и валидный запрос (200). Запуск:

```bash
npm run test:supabase
```

> Примечание: для некоторых функций нужны переменные `SUPABASE_URL` и `SUPABASE_SERVICE_ROLE_KEY`. Если не хочется прописывать их в системе, можно воспользоваться GitHub Actions (workflow `.github/workflows/edge-tests.yml` делает это автоматически).

## 3. Мониторинг логов Supabase

В `scripts/monitoring` лежит утилита на Deno, которая подтягивает логи Supabase и отправляет алерты в Slack/Teams или другой вебхук.

1. Создайте Supabase Access Token (Dashboard → Account → Access Tokens).
2. Получите `project ref` (видно в URL Supabase).
3. Настройте вебхук в Slack/Teams (или используйте любой HTTP‑endpoint).
4. Запустите:

```bash
SUPABASE_ACCESS_TOKEN=... \
SUPABASE_PROJECT_REF=abcd1234 \
ALERT_WEBHOOK_URL=https://hooks.slack.com/services/... \
MONITORED_FUNCTIONS="llm-chat,create-payment,jiva-embeddings-ingest" \
npm run monitor:logs
```

Скрипт можно положить в cron и запускать каждые 5 минут. Если появляются ошибки, он отправит сообщение с названием функции и текстом лога.

## Советы для оператора

1. **Перед релизом** — `npm run lint:auth && npm run test:supabase`.
2. **Во время работы** — держите мониторинг включённым (cron или отдельное окно терминала).
3. **При ошибке** — откройте Supabase → Logs → Edge Functions и передайте полный лог разработчику.

Это минимальный набор, который позволяет поддерживать сайт без глубоких технических знаний.
